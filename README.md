# Ultimate Sports Quiz Challenge

A simple sports trivia game built in Python.

---

### About the project
This program asks users a series of sports-related multiple-choice questions. As you go through the quiz, it keeps track of your score and shows your results at the end.

---

### Features
• Multiple-choice questions  
• Score tracking during the game  
• High score saved to a file  
• Option to replay  

---

### Running the program
Just run the Python file, type in your name, and answer each question by entering A, B, C, or D.

---

### Notes
I made this project to practice core Python concepts like loops, conditionals, functions, and file handling while building something interactive.
